# Face Mask Detection System | نظام كشف الكمامة

Computer Vision project to detect whether a person is wearing a face mask using images or live webcam feed.  
مشروع رؤية حاسوبية لكشف ما إذا كان الشخص يرتدي كمامة باستخدام الصور أو الكاميرا المباشرة.
